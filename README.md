# Spots

An image sharing site.

## Description

A webpage that allows users to share their photos with captions for the world. Functions inlcudes a like button, a edit profile button, and a button to add a new post.

## Tech Stack

- HTML
- CSS
- Responsive Design
- JavaScript
- Node.js
- NPM
- Webpack
- APIs
